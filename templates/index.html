What is ChillBot

ChillBot is a multipurpose Discord bot with everything you need. It can play any music, handle moderation, automated configurations, etc.

What does it do

ChillBot contains many features. Some features can be helpful for you. Read below for more information about some of the features.

Music

ChillBot can play anything you want. It can be K-POP, Rap, or any other music genres!

Moderation

ChillBot can handle moderation in your server! You can use the bot to kick, ban, or timeout the member(s). ChillBot also has an auto-mod feature located in the configurations.

And more...

ChillBot has many other features that you should try. What are you waiting for? Try it out!
